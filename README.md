# Freelance-Forge

Steps:
1. npm install to install all the dependencies from backend and frontend folder
2. make a .env files for backend and frontend and put these 

for backend .env file:

PORT=5000

MONGO_URI=MONGO_URI=make it from mongodb atlas

MESSAGE_MAC_KEY=your_message_mac_key

JWT_SECRET=your_jwt_secret

STRIPE_SECRET_KEY=your_stripe_secret_key

for frontend .env file:

REACT_APP_STRIPE_PUBLIC_KEY=your_stripe_public_key
